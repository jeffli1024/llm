# Goal
Build a character-level language model from scratch — no libraries beyond
PyTorch. Tokenize text, implement an embedding layer, write a basic attention
head by hand, and train the model to predict next characters on a small corpus
(e.g. Shakespeare). You will generate text before you read a single paper.
